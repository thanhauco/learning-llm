# MLOps & Deployment - BentoML, Modal, Replicate

## What You'll Learn

1. **BentoML** - ML model serving framework
2. **Modal** - Serverless GPU infrastructure
3. **Replicate** - Deploy models with one command
4. **CI/CD for ML** - Automated deployment pipelines

## Key Concepts

- Model versioning and registry
- A/B testing and canary deployments
- Auto-scaling and load balancing
- Cost optimization strategies
- Monitoring and observability

## Real-World Scenarios

- Deploy models in minutes, not days
- Auto-scale from 0 to 1000 GPUs
- Pay only for actual usage
- Roll back bad deployments instantly

## Tools Covered

- **BentoML:** Production ML serving
- **Modal:** Serverless compute
- **Replicate:** Managed model hosting
- **Ray Serve:** Scalable model serving
